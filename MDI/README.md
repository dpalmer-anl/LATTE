Runnng latte as an MDI engine 
=============================

# Compile LATTE

Source the variables in `vars`

  $ source vars 

Use the `makefile.CHOICHES.mdi` to compile. 

  $ cp makefile.CHOICHES.mdi ; make 

# Compile dummy driver

  $ cd ./MDI ; sh compile.sh 

# Run 

Run the driver as follws: 

  $ ./run_parseplice_new
